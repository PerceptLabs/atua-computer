# Owner Map by Track

## Track Owners

- Execution Core Track — Runtime/Engine team
- Host Integration Track — Bridge adapters team
- Agent Runtime Track — Orchestration API team
- QA/Compat Track — Validation and regression team

## Decision Ownership

- Phase gate decisions: Tech lead + QA lead
- Syscall prioritization: Execution Core + QA/Compat
- API breaking changes: Agent Runtime lead

## Cadence

- Weekly gate review with owners above.
- Any failed gate must publish mitigation owner + due date.
